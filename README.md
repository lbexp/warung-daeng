This project is served as a take home test assignment. This project has 2 services, the backend API service and the web client application service.

## Requirements

### Backend API

- NPM 10.7.0
- Node.js v18.20.4

### Web Client App

- NPM 10.7.0
- Node.js v18.20.4

## Tech stack

### Backend API

- NestJS
- PostgreSQL

### Web Client App

- VueJS
- Pinia
